### Installation:

    git clone git://github.com/samgomena/dotfiles.git

### Setup:

    chmod +x setup.sh
    ./setup.sh  

##### _Or for the bold_:

    git clone git://github.com/samgomena/dotfiles.git && chmod +x setup.sh && ./setup.sh


Note: This only installs a vimrc, it does **not** touch .bashrc or .bash_profile.
